KEY WARNINGS & SUCCESS FACTORS
⚠️ Common Pitfalls to Avoid
1.	Look-ahead bias: Ensure all backtests use point-in-time data only
2.	Overfitting: Use walk-forward optimization, not in-sample optimization
3.	Data leakage: Keep training/validation/test sets strictly separated
4.	Underestimating transaction costs: Always model slippage + fees realistically
5.	Ignoring correlations: The covariance matrix is CRITICAL for portfolio construction
