MLOps Portal Website
====================

This folder contains the MLOps portal website with links to MLflow, Prometheus, and Grafana.

FILES:
------
index.html - Main website file

HOW TO USE:
-----------
1. Open index.html in your web browser
2. Click on the service cards to open MLflow, Prometheus, or Grafana
3. Status badges show if services are online/offline

URLS:
-----
MLflow: http://localhost:5000
Prometheus: http://127.0.0.1:9090
Grafana: http://localhost:3000

FEATURES:
---------
- Modern dark theme design
- Status indicators for each service
- Direct links to all three tools
- Responsive design (works on mobile)
- Auto-refreshes status every 30 seconds
